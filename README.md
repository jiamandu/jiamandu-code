# Jiamandu Code

Dark-themed AI coding assistant for your terminal.

## Installation

```bash
# Unzip and install
unzip jiamandu-code.zip -d jiamandu-code
cd jiamandu-code
bash install.sh

# Set API key
export ANTHROPIC_API_KEY=sk-ant-your-key-here
```

## Usage

```bash
jiamandu              # Interactive session
jiamandu -p "prompt"  # Non-interactive
jiamandu --help       # Help
```

## Features

- Dark theme UI with cyan/teal accents
- Full Claude Code functionality
- All CLI tools and capabilities preserved
